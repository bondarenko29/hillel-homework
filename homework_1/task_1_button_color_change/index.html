Комбінації клавіш для VSCode ( MacOS || Windows )

  • Видалити рядок - Cmd + Shift + K || Ctrl + Shift + K
  • Копіювати лінію вгору або вниз - Opt + Shift + стрілка ( вгору або вниз ) || Shift + Alt + стрілка ( вгору або вниз )
  • Глобальний пошук - Cmd + Shift + F || Ctrl + Shift + F
  • Знайти - Cmd + F || Ctrl + F
  • Швидке відкриття - Cmd + P || Ctrl + P
  • Палітра команд - Cmd + Shift + P || Ctrl + Shift + P
  • Перемкнути бічну панель - Cmd + B || Ctrl + B
  • Перемикання вкладок - Cmd + Opt + стрілка (вліво або вправо) || Ctrl + ( Pg Up або Pg Down )
  • Редактор розділення - Cmd + \ || Ctrl + \
  • Переміщення між словами - Opt + стрілка ( ліворуч або праворуч ) || Ctrl + стрілка ( вліво або вправо )
  • Перехід до початку або кінця рядка - Cmd + стрілка ( вліво або вправо ) || Alt + стрілка ( вліво або вправо )
